Vim/Neovim Text Editor

Pendahuluan Pada kesempatan kali ini saya akan membahas Text Editor yang setiap hari saya gunakan, yaitu VIM (lebih tepatnya Neovim) Apa itu VIM? “Vim is a highly configurable text editor built to make creating and changing any kind of text very efficient.” - From vim.org VIM atau singkatan dari VI iMproved adalah sebuah Text Editor yang minimal, cepat, ringan, dan efisien namun sangat fleksibel. VIM diciptakan oleh Bram Moolenaar yang terinspirasi oleh Text Edit VI yang ada di Unix oleh Bill Joy....
