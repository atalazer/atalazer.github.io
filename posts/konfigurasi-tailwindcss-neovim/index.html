Konfigurasi Tailwind CSS di Neovim

Cara Mengkonfigurasi Neovim untuk Koding Tailwind CSS
May 28, 2021 · 2 menit · Atalazer | Suggest Changes

Pembukaan#

Pada kesempatan kali ini saya akan membagikan bagaimana saya melakukan konfigurasi untuk keperluan Web Development menggunakan Tailwind CSS di Neovim 0.5.

Pre-Requisite#

Sebelum melakukan konfigurasi, kita harus memenuhi beberapa syarat, antara lain:

  • tailwindcss-language-server
  • browser-sync (optional)

Instalasi#

# Install Language Server untuk tailwind css
npm install -g tailwindcss-language-server

# (Optional) browser-sync untuk live reload
npm install -g browser-sync

Konfigurasi#

Tambahkan Baris Berikut ke dalam konfigurasi LSP neovim kalian.

local configs = require("lspconfig/configs")
local util = require("lspconfig/util")
local nvim_lsp = require("lspconfig")

local server_name = "tailwindcss-language-server"

configs[server_name] = {
    default_config = {
        cmd = { server_name, "--stdio" },
        filetypes = { "css", "html", "svelte", "typescriptreact" },
        root_dir = util.root_pattern("tailwind.config.js", "package.json", ".git"),
        settings = {}
    },
    docs = {
        description = [[ ]],
        default_config = {
            root_dir = [[ ]],
        },
    },
}
nvim_lsp[server_name].setup({})

Penerapan#

  1. Pertama, kita buat directory baru untuk project yang akan digarap:
mkdir web-tailwind
cd web-tailwind
touch index.html
  1. Install tailwind-css:
npm init
npm install tailwindcss postcss postcss-cli autoprefixer
  1. Buat directory baru untuk menyimpan file .css dan buat file bernama tailwind.css:
mkdir -p src/{styles,scripts}
touch src/styles/tailwind.css
  1. Edit file tailwind.css dengan baris berikut:
@tailwind base;
@tailwind components;
@tailwind utilities;
  1. Buat dan edit file postcss.config.js pada root directory (tempat file index.html):
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  }
}
  1. Edit package.json pada root directory:
{
  ...
  "scripts": {
    "watch:css": "postcss src/styles/tailwind.css -o src/styles/style.css -w",
    "build:css": "postcss src/styles/tailwind.css -o src/styles/style.css"
  },
  ...
}
  1. Jalankan command npm run watch:css di root directory
  2. (Optional) Jalankan command browser-sync start --server --files . di root directory.
  3. Edit file index.html and Happy coding :) :
<!DOCTYPE html>
<html>
  <head>
    <title>Tailwind CSS</title>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width,initial-scale=1.0" />
    <link rel="stylesheet" href="/src/styles/style.css" />
  </head>
  <body></body>
</html>

Kesimpulan#

Nah itu dia, bagaimana cara saya Mengkonfigurasi Neovim untuk Proses Web Development menggunakan Framework Tailwind CSS. Terima Kasih :)
